# Thumbs de referência de estilo do canal

Coloque AQUI as capas (thumbnails) que servem de **padrão visual** do canal — as suas
melhores thumbs prontas. A **Etapa 6** (geração da capa) anexa **todas** as imagens desta
pasta como referência visual no Magnific, então **qualquer thumb gerada segue o estilo,
formato, enquadramento e iluminação destas aqui**.

## Como usar
1. Salve as imagens nesta pasta (`.png`, `.jpg`, `.jpeg`, `.webp`).
2. Nomeie de forma ordenável se quiser controlar a ordem (ex.: `ref_01.png`, `ref_02.png`…).
3. Pronto — na próxima vez que a Etapa 6 rodar, ela usa estas como base de estilo.

## Observações
- Não precisa ser exatamente as 5: quanto melhores e mais consistentes entre si, melhor o
  estilo travado. 3 a 6 boas capas costumam bastar.
- Estas referências definem **estilo/composição/luz**, NÃO os personagens — os personagens
  de cada vídeo vêm do prompt + fichas da Library (lock de personagem).
- A pasta é opcional: se estiver vazia, a thumb sai só pelo prompt + direção de arte (sem
  quebrar). Para mudar o local, use a env `LONGFORM_THUMB_REF_DIR`.
